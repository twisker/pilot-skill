# 路书导出说明

**中文** | [English](../en/guide-export.md)

PILOT 导出三种格式，内容同源（同一份 `itinerary.json` + `intake.json` + 参考游记索引），结构统一为**四段**。产物在 `~/.pilot/workspace/<trip-id>/exports/`。

| 格式 | 定位 | 特点 |
|------|------|------|
| **Excel**（.xlsx） | 出行中实用工具 | **三者中最完整**：4 sheet、费用 SUM/SUMIF 公式、类别下拉校验、跨表引用 |
| **PDF** | 打印 / 转发 | 每段独立起页，静态呈现，模板表留白供打印后手写 |
| **Word**（.docx） | 二次编辑 | 与 PDF 同结构，标题层级规整，方便你自己继续改 |

## 四段结构（Excel 的 4 个 sheet 与之一一对应）

### 封面（四段之外，最前）

路书标题（`目的地 X天交通方式路书`）、出行日期、出行人（X大人X小孩X老人）、出发地、生成信息。

### 一、行程安排及费用预算

**主表：一天一行**——天序/日期、行程摘要（当天景点串联，如「赛里木湖 → 果子沟大桥 → 伊宁」）、住宿地、分类费用列（门票/餐饮/住宿/交通/其他，按实际用到的类目动态建列）、当日合计；末行总计。

- **与预算对比**：总计 vs 你的预算，超预算标红；人均 = 合计 ÷ 出行人数；未计价条目数会提示（避免「看起来便宜其实没算全」）
- **附注小节**：住宿汇总（入住日/名称/价格/预订链接 + 小计）、交通汇总（大交通 + 每日路段 + 自驾取还车附注）、餐饮推荐（按天分组）

Excel 版此 sheet 的总计行是**活公式**（SUM），你改任何一格费用，总计与预算对比自动更新。

### 二、每日详细行程

每天一节，逐条目列出：时间（未定显示「待定」）、类型（景点/餐饮/住宿/交通/其他）、名称与备注、坐标（查证过的才有）、费用、预订入口（只给官方直链，查不到不编）。

### 三、出行必备自检清单（模板 sheet）

**这是模板，供你打印后手工勾选**。事项由确定性规则从你的行程画像推导（不是 AI 现编的）：

- 基础：证件、充电、药品
- 季节推导：出行月份 → 衣物建议（如 7-8 月新疆：昼夜温差、防晒、冲锋衣）
- 交通推导：自驾 → 行车记录仪/拖车绳/备用油桶提示
- 人群推导：带小孩 → 儿童用品；带老人 → 常用药提示
- 偏好推导：摄影 → 器材项

表格列：类别 | 事项 | 责任人（空）| 出发前 ☐ | 出行中 ☐，末尾留空白行供追加。

**应急信息**并入本段末尾：110/120/122/119、目的地注意事项（如新疆：边防证、信号盲区、加油站间距）、道路救援模板位、行程数据备份路径。

### 四、消费流水及费用核算（模板 sheet）

**出行期间记账用**：

- **消费流水表**：日期 | 项目 | 类别 | 金额 | 付款人 | 备注。PDF/Word 预留约 20 行空白手写；**Excel 版带类别下拉校验**，直接在手机/电脑上记
- **费用核算小表**：类别 | 预算数 | 实际 | 差额。「预算数」列是**真实值**（从第一段的分类合计跨表引用而来）；「实际」「差额」留空，出行结束后填写。Excel 版「实际」列用 SUMIF 自动从流水表按类别汇总——只要流水记了账，核算表自动出结果

### 附录（最后）

参考游记来源：本次路书参考的精选游记列表（标题/平台/链接），行程可溯源。

## 降级策略

- Excel 或 Word 导出失败 → 保证 PDF 出来，失败的格式如实告知
- PDF 也失败 → PILOT 会解释原因并提供「修复重试」或「先给 Markdown 版顶用」两个选项
